ZeroQuest
position-is-quest · procedural narrative mission system · v1.0.0
O(1) generation zero storage deterministic

The position is the quest. ZeroQuest uses the objective's world coordinate as the only input — hashing it with xxhash32([x, y, z], worldSeed) to deterministically derive quest type, timer, narrative briefing, stage locations, and outcome text. No database. No file. No server call.

Enter any coordinate below and click Generate Quest. Change any single digit and a completely different mission appears — yet return to the same coordinate and the same quest will always be there, waiting.
